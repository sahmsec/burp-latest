# burp-latest

## Installation Commands

### Linux

```bash
wget -qO BurpLoaderKeygen.jar https://github.com/uppermo0n/burp-latest/releases/download/v1/BurpLoaderKeygen.jar && java -jar BurpLoaderKeygen.jar
```

### Windows (PowerShell)

```powershell
iwr https://github.com/uppermo0n/burp-latest/releases/download/v1/BurpLoaderKeygen.jar -OutFile BurpLoaderKeygen.jar; java -jar BurpLoaderKeygen.jar
```

### macOS

```bash
curl -Ls -o BurpLoaderKeygen.jar https://github.com/uppermo0n/burp-latest/releases/download/v1/BurpLoaderKeygen.jar && java -jar BurpLoaderKeygen.jar
```
